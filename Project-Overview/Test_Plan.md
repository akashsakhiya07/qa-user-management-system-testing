## 1. Introduction
This test plan describes the testing approach, scope, resources, and schedule for testing the User Management System web application.
The purpose of this document is to ensure that the application meets business requirements and delivers a stable user experience.

---

## 2. Scope of Testing

### In Scope
- User Registration
- User Login
- User Logout
- Input validations
- Admin view of registered users
- User status management (Active / Inactive)
- UI and functional testing
- Smoke and regression testing

### Out of Scope
- Payment related features
- Email verification
- Password recovery
- Performance and security testing

---

## 3. Test Approach
- Manual testing will be performed for all in-scope functionalities
- Test cases will be designed based on business requirements
- Bugs will be logged with proper severity and priority
- Re-testing and regression testing will be done after bug fixes

---

## 4. Test Environment
- Application Type: Web Application
- Browser: Chrome
- Operating System: Windows
- Test Data: Dummy user data

---

## 5. Test Deliverables
- Test Plan
- Test Cases
- Bug Reports
- API Testing Notes
- Database Validation Notes
- Test Execution Report
- Test Summary Report

---

## 6. Entry Criteria
- Application build is available for testing
- Test environment is accessible
- Requirements are finalized

---

## 7. Exit Criteria
- All planned test cases are executed
- Critical and high severity bugs are fixed
- Application is stable for basic usage

---

## 8. Risks and Mitigation

| Risk | Mitigation |
|-----|-----------|
| Incomplete requirements | Clarify with stakeholders |
| Frequent build changes | Perform regression testing |
| Time constraints | Prioritize critical features |

---

## 9. Roles and Responsibilities
- QA Tester: Test case creation, execution, bug reporting
- Developer: Bug fixing
- Project Stakeholder: Requirement clarification and sign-off

---

## 10. Approval
This test plan is prepared to demonstrate a real-world QA testing approach for the User Management System.
